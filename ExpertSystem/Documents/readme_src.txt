Un répertoire src contenant les sources de votre système, avec un fichier
readme.txt expliquant comment générer l’exécutable à partir du code source. Si
pertinent, vous pouvez ajouter un fichier utilisation.pdf expliquant comment
utiliser votre système. Par exemple, si la spécification de votre base de
connaissance se fait par une interface non trivial, c’est ici que vous mettriez les
instructions.
